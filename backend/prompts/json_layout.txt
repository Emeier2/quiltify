RULES:
1. Blocks must not overlap — no two blocks may share any grid cell.
2. Blocks must cover EVERY cell in the grid — the union of all blocks must equal width × height.
3. Use only rectangles (axis-aligned, integer coordinates, width ≥ 1, height ≥ 1).
4. x ranges from 0 to grid_width-1; y ranges from 0 to grid_height-1.
5. x + width ≤ grid_width; y + height ≤ grid_height.
6. Each fabric id must appear in at least one block.
7. Design the pixel arrangement so the overall silhouette of the subject is recognizable from the color masses — use the main subject color for the subject body area, a contrasting background color for negative space.
8. Use generalized solid color names that match the design (e.g., for an owl: "Warm Brown" body, "Cream" face, "Gold" eyes, "Navy Solid" background). Use 4-8 distinct colors with clear roles: subject body, detail/accent, and background.
